MODULE 2 — DistFlow Solver Analysis
=====================================
File: module2.m (distflow_solver_analysis.m)

WHAT IT DOES
------------
Runs a full DistFlow (backward-forward sweep) baseline analysis on the
IEEE 33-bus network including:
  - Baseline single-snapshot load flow
  - 24-hour time-series load flow using the hourly multiplier profile
  - Stress tests at load multipliers [1.0, 1.2, 1.4]
  - Manual VAR (reactive power) support experiments

DistFlow is an iterative power flow method for radial networks.
It does NOT require YALMIP or Gurobi — uses built-in MATLAB only.

WHY RUN IT
----------
Establishes baseline power flow results before OPF.
Shows where voltage violations occur under normal and stressed loading.

OUTPUT
------
Saves results to ./out_distflow/ (voltage profiles, loss summaries).

DEPENDS ON

build_distflow_topology_from_branch_csv.m
