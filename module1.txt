MODULE 1 — Topology Construction Verification
==============================================
File: module1.m (topology_construction_verification.m)

WHAT IT DOES
------------
Verifies the IEEE 33-bus radial network topology was built correctly.
Calls build_distflow_topology_from_branch_csv.m and checks:
  - Total buses = 33, total tree branches = 32 (nb-1)
  - Root bus = 1 (slack)
  - No duplicate or missing branch connections
  - Dropped tie-lines listed (non-tree branches removed for radial BFS)

WHY RUN IT
----------
Run once at the start to confirm the CSV data was read correctly
before using the topology in any OPF or DistFlow module.

OUTPUT
------
Prints verification report to console. No files saved.

DEPENDS ON
----------
build_distflow_topology_from_branch_csv.m
mp_export_case33bw/branch.csv
