=============================================================
WHAT THE EXISTING CODE DOES
=============================================================

The repository implements an IEEE 33-bus radial distribution
network analysis toolkit, progressing from basic DistFlow
load-flow to SOCP-OPF with Electric Spring smart-load control.

Module progression (original commits):
  Module 1-3  : Load profile and topology builders
  Module 7    : 24-hour SOCP OPF with Qg reactive support  (optimization.m)
  Module 8    : ES smart-load SOCP OPF  (run_socp_opf_24h_yalmip_gurobi_es.m)
  Framework   : Multi-scenario ES runner  (run_es_scenario_framework.m)

=============================================================
CURRENT ES MODEL
=============================================================

The ES is modelled at OPF level as a controllable NCL fraction:

  P_load(j,t) = P_CL(j,t) + u(j,t) * P_NCL(j,t)
  Q_load(j,t) = Q_CL(j,t) + u(j,t) * Q_NCL(j,t)

  where P_CL = (1-rho)*Pd,  P_NCL = rho*Pd
  and   u_min <= u(j,t) <= 1

ES parameters:
  rho_val    — NCL fraction (same for all ES buses, scalar)
  u_min_val  — minimum NCL service factor
  lambda_u   — curtailment penalty in objective

The NCL fraction rho is the same for all ES buses in the
current code. The new research extends this to bus-specific
rho_j (ES sizing variable) and adds binary placement z_j.

=============================================================
KEY CONSTRAINT STRUCTURE (DistFlow SOCP)
=============================================================

Variables per branch (k) and time (t):
  P(k,t)   active power flow
  Q(k,t)   reactive power flow
  l(k,t)   squared current magnitude
  v(j,t)   squared voltage magnitude at each bus j

Power balance at bus j:
  P(parent->j, t) = P_load(j,t) + sum_k P(j->child_k, t) + R(j)*l(j,t)

Voltage drop:
  v(j,t) = v(parent,t) - 2*[R*P + X*Q](parent->j,t) + (R^2+X^2)*l(j,t)

SOCP relaxation cone:
  [2*P; 2*Q; l-v_parent] norm <= l + v_parent

Voltage limits:
  Vmin^2 <= v(j,t) <= Vmax^2
  v(root,t) = 1.0  (slack bus)
